# 因子小侦探：细菌分拣实验室

这是一个给小学生设计的前端小游戏，升级为**细菌聚类实验**：

- **6 因子观察**：轮廓、连接方式、主色调、速度、荧光、表面颗粒。
- **3 因子决定物种**：只有轮廓 + 连接方式 + 主色调参与真实分类。
- **3 因子是干扰项**：速度、荧光、表面颗粒用于迷惑，训练抗干扰能力。
- **无标签分区**：培养皿分区不贴标签，先独立判断，再点“检查答案”。

## 怎么玩

1. 点击“新一关”。
2. 观察上方一大块无分割区域（约半屏）里自由移动、旋转的细菌样本，先点击选中，再点击下方分区投放。
3. 如果投错了，选中该细菌后点击上方主区域可释放回去。
4. 点击“检查答案”，查看正确率和反馈（错误隔离箱会警告且内部微生物躁动）。
5. 点“给我提示”可获得思路提示（不会直接告诉你标签）。

## 运行

```bash
python3 -m http.server 4173
```

打开 `http://localhost:4173`。

## 常见问题

如果你看到类似下面的报错：

```
Unsafe attempt to load URL file:///... 'file:' URLs are treated as unique security origins
```

说明你是直接双击 `index.html` 用 `file://` 打开的。请改为：

1. 在项目目录运行 `python3 -m http.server 4173`
2. 浏览器访问 `http://localhost:4173`

这样可以避开浏览器对本地文件的同源策略限制。
