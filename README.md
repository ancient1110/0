# 因子小侦探：水果分拣大挑战

这是一个给小学生设计的前端小游戏，主打：

- **因子拆分**：把问题拆成颜色、大小、纹理三个因子。
- **细粒度特征**：不是只看“这像什么”，而是看更细的小特征。
- **分类思维**：通过拖拽分拣训练规则意识和观察力。

## 怎么玩

1. 点击“新一关”。
2. 把水果拖到对应的“颜色+大小+纹理”分拣箱。
3. 点击“检查答案”，查看正确率和反馈。
4. 点“给我提示”可获得单个样本的因子提示。

## 运行

```bash
python3 -m http.server 4173
```

打开 `http://localhost:4173`。
